Day #01

¿Que hice?

En el primer dia de este reto, lo utilicé para crear toda la página web, haciendola responsiva y demás cosas, para crear un sitio donde subir mi progreso y que de esta forma lo pueda de alguna forma medir con el paso del tiempo, agregando el tiempo que utilicé para el proceso de cada cosa que haga. Esto lo utilizaré como blog personal, con el objetivo de cumplir el reto y motivarme a mi mismo a hacerlo posible

Se practicó...

Practique tecnicas de media queries, displays, algo de JavaScript, pero realmente solo lo copie, por que todavia no se mezclar JavaScript con html y css. Además, practiqué algo de diseño con algo de ayuda de chelsy y yaks, a seguir con la vida y viendo mas cursos

Day #02

¿Que hice?

El día de hoy, el proyecto que realicé, fue una calculadora de peso según la gravedad de otros planetas, esto totalmente inspirado de unos videos del curso de programacion basica de Platzi. Este proyecto funciona muy bien como una pequeña practica para evaluar lo que he aprendido sobre JavaScript.

Se practicó...

Practiqué ciertas cosas como tipos de variables, algo de DOM, uso de operadores (aunque estos al final no pude usarlos porque me generaban un error en el programa) variables y algunas operaciones dentro del JavaScript
